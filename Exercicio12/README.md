Faça um procedimento que receba duas notas parciais de um aluno. O programa deve calcular a média alcançada por aluno e apresentar:
a.	A mensagem "Aprovado", se a média alcançada for maior ou igual a 10;
b.	A mensagem "Reprovado", se a média for menor do que 10;
c.	A mensagem "Aprovado com Distinção", se a média for igual a 20.

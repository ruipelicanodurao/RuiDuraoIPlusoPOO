Escreva um programa que pergunte ao utilizador qual o PIN que tem de introduzir para que possa receber a mensagem "PIN CORRETO". Em todas as outras tentativas erradas, deverá mostrar "PIN ERRADO".
a)	Tip: Defina o PIN internamente (p.ex int pin = 1234;)
b)	Tip2: Esteja "constantemente" a verifique se o input do utilizador é errado (ou seja diferente de "pin").
